# Inline Styles, Basic Interaction, and Server-Side Validation

A small Node.js and Express project that demonstrates inline styling, basic client-side interaction, and server-side form validation.

## Features
- Responsive form UI with inline styles
- Basic client-side validation using inline JavaScript
- Server-side validation for submitted form data
- Temporary server-side storage of validated submissions

## Setup
1. Install dependencies:
   npm install
2. Start the server:
   npm start
3. Open your browser at:
   http://localhost:3000
